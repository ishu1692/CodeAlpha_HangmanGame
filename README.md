# 🎮 Python Hangman Game

An enhanced console-based Hangman Game developed using Python.  
The player guesses letters to reveal a hidden word before the hangman is completed.

## ✨ Features
- Random word selection
- Hints for each word
- ASCII Hangman graphics
- Input validation
- Win/Lose conditions
- Interactive terminal gameplay

## 🛠 Technologies Used
- Python 3

## ▶ How to Run

```bash
python hangman.py
